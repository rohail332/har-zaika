# 🍽️ Har Zaika - Recipe Website

Official website for **Har Zaika** featuring recipes from Asian, Chinese, Vegetarian, Desserts, Fast Food and Drinks categories.

</div>

## Tech Stack

<div align="center">

[![My Skills](https://skillicons.dev/icons?i=html,scss,js)](https://skillicons.dev)

</div>

- **HTML5** - Structure and markup
- **SCSS** - Preprocessed CSS compiled to `dist/main.css`
- **JavaScript** - Interactive navigation

## Features

- Recipe categories with 28+ detailed recipes
- Health blog with Harvard Health articles
- Newsletter subscription
- Responsive design with mobile navigation
- Team section and contact form

## Files

- `index.html` - Homepage
- `recipes.html` - Recipe collection
- `about.html` - Team information
- `blog.html` - Health articles
- `contact.html` - Contact form
- `script.js` - Navigation functionality
- `img/` - Image assets
- `Recipes/` - Individual recipe pages
